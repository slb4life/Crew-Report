# Crew-Report

Crew Report is a standalone SwiftUI iPhone app for tracking construction crew hours, live shifts, payroll, reviews, exports, backups, widgets, and iCloud sync.

- Privacy policy: [PRIVACY_POLICY.md](PRIVACY_POLICY.md)

## Highlights

- Project-based crew hours, payroll, reviews, and history.
- Live shift tracking with break minutes and active-shift status.
- Weekly and daily delivery flows for crew members and supervisors.
- PDF, CSV, Plain Text, and Simple Text exports.
- Backups, local snapshots, widgets, and iCloud sync.
